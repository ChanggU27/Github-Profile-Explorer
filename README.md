# GitHub Profile Explorer

A React application for searching GitHub users and exploring their repositories.

## Features

- Search public GitHub profiles
- View profile statistics
- Search repositories
- Sort repositories by stars or updates
- Dark mode support
- Mobile-responsive
- Pagination
